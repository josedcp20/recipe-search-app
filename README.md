## Project Description
A web application for searching meal recipes by ingredients.
Users can search, view details, and save favorite recipes.

## Target Audience
Pteranodon is designed for users who are interested in discovering, organizing and experimenting with meat-based recipes

## Cuisine Focus
Meat-based dishes

## How to Run
1. Download this repository
2. Open index.html in browser
3. No installation needed!
